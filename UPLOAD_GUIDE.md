# 📤 Загрузка данных через веб-интерфейс

## Обзор

Теперь вы можете загружать Excel файлы напрямую через веб-интерфейс без необходимости запускать Python скрипты вручную!

## 🎯 Как использовать

### 1. Откройте страницу администрирования

Перейдите на [http://localhost:3000/admin](http://localhost:3000/admin) или нажмите на карточку **"Администрирование"** на главной странице.

### 2. Выберите тип данных

- **Sales** - основные данные продаж (2020-2025)
- **Ready Sales** - данные с клиентами (2024-2025)

### 3. Загрузите Excel файл

- Нажмите на область загрузки или перетащите файл
- Поддерживаются форматы: `.xlsx`, `.xls`

### 4. Нажмите "Загрузить в базу данных"

Система автоматически:
- Прочитает Excel файл
- Проверит структуру
- Загрузит данные в базу
- Покажет результат (количество загруженных записей или ошибки)

## 📋 Требования к структуре файлов

### Sales (основные продажи)

**Обязательные колонки:**
- `КОД_ТОВАРА` - код товара
- `Дата` - дата продажи

**Опциональные колонки:**
- `ГРУППА_ТОВАРА` - группа товара
- `РЕГИОН` - регион
- `СКЛАД` - склад
- `СЧЕТЫ` - счета
- `ДОПОЛН__КОЛ-ВО` - количество
- `ТОВАРЫ` - название товара
- `ЦВЕТ` - цвет
- `профиль_перечень` - профиль

**Пример структуры:**
```
КОД_ТОВАРА | ГРУППА_ТОВАРА | РЕГИОН | СКЛАД | Дата       | ДОПОЛН__КОЛ-ВО | ТОВАРЫ
BKT-70     | Профили       | Ташкент | Юнусабад | 2025-01-15 | 1250.5        | ALDOKS NEO
```

---

### Ready Sales (продажи с клиентами)

**Обязательные колонки:**
- `Дата` - дата продажи
- `Клиент` - название клиента

**Опциональные колонки:**
- `Дилер` - дилер
- `Клиент_ИД` - ID клиента
- `Инвоиcе_CИД` - номер инвойса
- `Тип` - тип
- `Тип_организации` - тип организации
- `Продукт` - продукт
- `Группа_продуктов` - группа продуктов
- `Количество` - количество
- `Вес_кг` - вес в кг
- `Общая_сумма` - сумма
- `Доход` - доход
- `Валюта` - валюта
- `ТОВАРЫ` - товары

**Пример структуры:**
```
Дата       | Дилер    | Клиент           | ТОВАРЫ     | Вес_кг | Общая_сумма
2025-01-15 | ООО Akfa | Илхом ака (+998) | ALDOKS NEO | 9918.78| 45000000
```

## 🛠️ Установка зависимостей

Перед первым использованием установите необходимые библиотеки:

```bash
cd backend
pip install -r requirements.txt
```

Новые зависимости:
- `pandas` - для обработки Excel файлов
- `openpyxl` - для чтения .xlsx файлов

## 🔧 Технические детали

### Backend API

**Endpoint:** `POST /api/upload/excel/`

**Параметры:**
- `file` (file) - Excel файл
- `data_type` (string) - тип данных (`sales` или `ready_sales`)

**Ответ при успехе:**
```json
{
  "message": "Успешно загружено 1500 записей",
  "records_count": 1500,
  "errors": null
}
```

**Ответ при ошибке:**
```json
{
  "error": "Отсутствуют обязательные колонки: КОД_ТОВАРА",
  "available_columns": ["Код", "Дата", "Товар"]
}
```

### Процесс загрузки

1. **Валидация файла**
   - Проверка формата (.xlsx или .xls)
   - Проверка наличия обязательных колонок

2. **Обработка данных**
   - Чтение Excel файла с помощью pandas
   - Парсинг каждой строки
   - Конвертация типов данных

3. **Сохранение в БД**
   - Использование транзакций (все или ничего)
   - Пропуск пустых строк
   - Обработка ошибок

4. **Отчет о результатах**
   - Количество успешно загруженных записей
   - Список ошибок (до 10 первых)

## ⚠️ Важные примечания

### Производительность
- Загрузка больших файлов (>10000 строк) может занять время
- Во время загрузки не закрывайте страницу
- При загрузке очень больших файлов (>50000 строк) рекомендуется разбить на части

### Дубликаты
- **ВНИМАНИЕ:** Система не проверяет на дубликаты автоматически
- Повторная загрузка того же файла создаст дубликаты
- Перед загрузкой убедитесь, что данные еще не в базе

### Ошибки
- Если загрузка прервалась с ошибкой, изменения откатываются (транзакция)
- Система покажет первые 10 ошибок для анализа
- Проверьте формат данных в проблемных строках

## 🔍 Отладка

### Файл не загружается?
1. Проверьте формат файла (.xlsx или .xls)
2. Убедитесь, что есть обязательные колонки
3. Проверьте, что backend сервер запущен
4. Откройте консоль браузера (F12) для детальных ошибок

### Неправильные данные?
1. Проверьте названия колонок (точное совпадение)
2. Убедитесь в правильности формата дат
3. Проверьте числовые поля на корректность

### Медленная загрузка?
1. Разбейте большой файл на части
2. Удалите ненужные колонки из Excel
3. Проверьте нагрузку на сервер
