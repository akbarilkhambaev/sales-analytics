## 🎯 Особенности:

### 1. Real-time фильтрация:
При изменении любого фильтра данные обновляются автоматически через API.

### 2. Расчет ИТОГОВ:
Итоговая строка считается на клиенте, суммируя все товары/группы.

### 3. Проценты роста:
Формула: `((текущий год - прошлый год) / прошлый год) * 100`

Полная типизация для безопасности кода.

## 🐛 Возможные проблемы:

### CORS ошибки:
Django настроен для localhost:3000. Если Next.js запустился на другом порту, обновите в `backend/config/settings.py`:
```python
CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
    "http://localhost:3001",  # Добавьте этот
]
```

### Port занят:
Если порт 3000 занят, Next.js автоматически использует 3001.
